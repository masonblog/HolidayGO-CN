# HolidayGO-CN

> 中国大陆居民假期政策一站式查询工具

**HolidayGO-CN** 帮助你在几十秒内查清自己所在省份的七类法定假期权益：年休假、婚假、产假、陪产/护理假、育儿假、探亲假、病假。

国家层面的统一规定与各省（自治区、直辖市）的特殊规定自动合并展示，让你不用翻遍法规文件，也能快速获得准确、可溯源的假期信息。

---

## 覆盖范围

| 假期类型 | 说明 |
|---|---|
| **年休假** | 按累计工作年限确定 5～15 天带薪假期 |
| **婚假** | 法定 3 天 + 各地奖励性延长天数 |
| **产假** | 正常生育、难产、多胞胎、流产等情形 |
| **陪产/护理假** | 男方在配偶生育期间的带薪陪护假 |
| **育儿假** | 子女三周岁前父母每年可休的带薪假期 |
| **探亲假** | 与父母或配偶分居两地的职工探亲待遇 |
| **病假** | 因病或非因工负伤的医疗期及病假工资标准 |

## 如何使用

### 1. 地图导航
打开首页，点击中国地图上的省份，即可进入该地区的假期详情页。地图中以不同底色区分「已有省级特别规定」和「沿用中央规定」的地区。

### 2. 列表浏览与搜索
首页下方的列表按行政区分组，支持关键词搜索。输入省份名称（如"浙江"、"广东"）即可快速定位。

### 3. 查看详情
进入地区页面后，七类假期以标签页形式展示。每条规则均标注来源：
- **中央** — 国家法律法规的一般规定
- **本地** — 本省地方性法规、条例或实施办法

每条规则下方均附有原文链接（全国人大法律数据库、中国政府网、人社部及地方官网），方便你进一步核实。

## 数据来源

本站数据以权威官方渠道为基础，所有条文均尽量给出原始出处：

- [国家法律法规数据库](https://flk.npc.gov.cn)（全国人大）
- [中国政府网](https://www.gov.cn)（国务院文件）
- [人力资源和社会保障部](https://www.mohrss.gov.cn)
- 各省（自治区、直辖市）人大及政府官网

## 免责声明

本站内容仅供信息参考，**不构成法律意见**。法律法规可能随时修订，各地实施细则也存在差异，具体待遇请以：
- 最新生效的法律法规
- 你的劳动合同
- 用人单位规章制度
- 当地人社部门的解释与执行口径

为准。涉及具体个案或权益争议，建议咨询专业律师或当地劳动仲裁机构。
